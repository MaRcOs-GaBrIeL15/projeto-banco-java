# Sistema Bancário

Projeto desenvolvido em Java com o objetivo de simular um sistema bancário simples, permitindo o gerenciamento de clientes, criação de contas e realização de operações bancárias.

## 📌 Sobre o projeto

O sistema permite que um cliente seja cadastrado e possa criar diferentes tipos de contas bancárias. Cada conta possui suas próprias características e regras, utilizando conceitos de Programação Orientada a Objetos (POO), como herança, encapsulamento e polimorfismo.

## 🚀 Funcionalidades

- Cadastro de clientes;
- Criação de contas bancárias;
- Escolha do tipo de conta no momento da criação:
  - Conta Corrente;
  - Conta Especial;
  - Conta Poupança;
- Acesso às contas cadastradas;
- Realização de operações bancárias:
  - Depósito;
  - Saque;
  - Consulta de informações da conta;
- Registro de movimentações realizadas;
- Controle da situação da conta.
